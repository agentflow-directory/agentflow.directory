    # AgentFlow Healthy Routine Builder

    Open-source, no-code starter agent for **AgentFlow.Directory**.

    AgentFlow Healthy Routine Builder helps users improve **routines** as part of a broader **Personal Operating System for Health**.

    This starter pack is designed for creators who want a practical Custom GPT template with little or no coding.

    ---

    ## What this agent does

    It helps the user turn vague health intentions into a practical system they can actually follow.

    Typical outcomes include:
    - morning routine
- evening routine
- full-day routine map
- weekday structure
- maintenance checklist

    ---

    ## Core positioning

    This is not a medical diagnostic tool and not a replacement for licensed care.

    It is a **guided health-organization agent**:
    - practical instead of preachy,
    - structured instead of chaotic,
    - sustainable instead of extreme,
    - system-focused instead of motivation-only.

    It fits into the AgentFlow concept of a **Personal Operating System for Health**.

    ---

    ## Best use cases

    - Create a healthy morning routine
- Build a full-day health-supportive routine
- Turn scattered habits into one system
- Stabilize weekdays
- Make health easier to maintain

    ---

    ## Best audience

    - busy professionals

    ---

    ## No-code MVP setup

    You can build this as:
    1. a **prompt-only Custom GPT**, or
    2. a **Custom GPT + one knowledge file** with templates, checklists, and examples.

    Minimal setup:
    - name: AgentFlow Healthy Routine Builder
    - paste the system prompt from `system-prompt.md`
    - optionally upload a small knowledge file with frameworks and reusable templates

    ## Notes

    This agent is best used inside ChatGPT as a guided planning and coaching assistant.
    It does not automatically sync wearables, medical systems, or third-party health apps.

    For a future advanced version, you could add:
    - reusable weekly templates,
    - printable checklists,
    - integrations with journaling or task tools,
    - dashboard exports for personal tracking.
