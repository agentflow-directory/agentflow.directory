# AgentFlow Idea Clarifier

Open-source, no-code starter agent for **AgentFlow.Directory**.

AgentFlow Idea Clarifier helps a user turn a rough idea into a structured agent specification that can be:
- used as a Custom GPT in ChatGPT,
- published as a listing in AgentFlow.Directory,
- documented and forked from GitHub.

This starter pack is intentionally minimal. It is designed for creators who want a working agent example with **little or no coding**.

---

## What this agent does

A user can come in with a vague prompt such as:
- “I want an agent that helps qualify leads from my website.”
- “I need a GPT that turns my notes into LinkedIn posts.”
- “Help me define an internal support assistant for my small business.”

The agent then turns that into a clear package with:
- agent name,
- one-line pitch,
- target user,
- jobs to be done,
- inputs and outputs,
- recommended workflow,
- no-code MVP setup,
- limitations and guardrails,
- draft listing text for AgentFlow.Directory.
