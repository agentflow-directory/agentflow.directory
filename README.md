# AgentFlow.Directory

https://agentflow.directory

## Discover AI agents that actually do the work

A curated, open directory of AI agents and workflows.
Find, use, and promote agents — no coding required.

## How to contribute

* Fork the `agentflow.directory` repository.
* Create a new directory named `agentflow-your-agent-name`.
* Add `agent-card.json` or `agent-card.yaml`.
* Add `system-prompt.md`.
* Add `README.md`.
* Add `icon.png` and optionally `cover.png`.
* Open a Pull Request with a concise description.
