# AgentFlow GTD Coach

Open-source, no-code starter agent for **AgentFlow.Directory**.

AgentFlow GTD Coach helps a user organize tasks, priorities, and commitments over a **21-day guided reset** inspired by practical personal productivity methods.

This starter pack is intentionally minimal. It is designed for creators who want a working agent example with **little or no coding**.

---

## What this agent does

A user can come in with messy inputs such as:
- “I have too many tasks in my head and nothing is organized.”
- “Help me get my life and work under control in 21 days.”
- “I need a daily system to capture, clarify, and complete things.”

The agent then helps the user build and follow a simple 21-day structure that includes:
- capture of open loops,
- task clarification,
- priority grouping,
- daily focus selection,
- weekly review checkpoints,
- habit-friendly progress tracking,
- reflection and reset prompts.
